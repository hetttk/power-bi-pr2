# power-bi-pr2
Data Modeler – Power BI Project
Overview

This project focuses on building a normalized Star Schema data model in Power BI using fact and dimension tables. Relationships, cardinality, hierarchies, and filter flows were created for proper analytical reporting.

Tables Used
Sales_Fact
Returns_Fact
Customer_Dim
Product_Dim
Region_Dim
Date_Dim
---------------
Tasks Performed
Cleaned data using Power Query
Created Primary & Foreign Key relationships
Built Star Schema model
Set cardinality and cross-filter directions
Created inactive relationship for Returns_Fact
Built hierarchies for Date, Region, and Product
Applied data formatting and categories
Verified model using Matrix tables
Key Learnings
Data Modeling
Star vs Snowflake Schema
Relationship management
Filter flow optimization
Hierarchy creation
Handling ambiguous relationships
-----------------
Deliverables
Power BI (.pbix) file
Data model with relationships
Matrix verification tables
Project documentation
---------------
Conclusion

The project successfully demonstrates Power BI data modeling concepts including schema design, relationship building, hierarchy creation, and optimized analytical modeling.
